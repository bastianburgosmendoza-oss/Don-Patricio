Conexión Supabase
Configura tu base de datos gratuita para recibir reportes online de los vecinos
Sin configurar
⚡ Configuración en 3 pasos:
1. Entra a supabase.com → Crea una cuenta gratuita → "New project"
2. En tu proyecto: SQL Editor → ejecuta el script de abajo → luego Settings → API y copia la URL y anon key
3. Pega los datos aquí → guarda → comparte el link con los vecinos
-- Ejecuta esto en el SQL Editor de Supabase:
CREATE TABLE lecturas_pendientes (
  id bigint generated always as identity primary key,
  parcela_id integer,
  parcela_nombre text,
  propietario text,
  mes integer,
  anio integer,
  lecturas jsonb,
  estado text default 'pendiente',
  enviado_en timestamptz default now()
);
ALTER TABLE lecturas_pendientes ENABLE ROW LEVEL SECURITY;
CREATE POLICY "insert_public" ON lecturas_pendientes FOR INSERT WITH CHECK (true);
CREATE POLICY "select_anon" ON lecturas_pendientes FOR SELECT USING (true);
CREATE POLICY "update_anon" ON lecturas_pendientes FOR UPDATE USING (true);
